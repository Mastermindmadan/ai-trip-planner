# SmartTrip AI - Intelligent Travel Planner

SmartTrip AI is a modern, AI-powered travel planning application built with React, Tailwind CSS, and Framer Motion. It simplifies the complexity of itinerary planning by providing curated, optimized travel plans based on user preferences.

## 🚀 Features

- **AI-Powered Itineraries**: Generate personalized travel logs based on destination, duration, budget, and travel style.
- **Dynamic Dashboard**: Interactive interface for planning and visualizing your next adventure.
- **Trip Archive**: Securely store and manage your past trip histories.
- **Theme Support**: Seamless transitions between Light and Dark modes.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Real-time Animations**: Smooth, "travel-vibe" transitions and interactions powered by Framer Motion.
- **Mock Authentication**: Secure user session simulation (GitHub-ready for integration with any Auth provider).

## 🛠️ Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (motion/react)
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **Deployment**: Optimized for Cloud Run and GitHub Pages.

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/smarttrip-ai.git
   ```

2. Navigate to the project directory:
   ```bash
   cd smarttrip-ai
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000`.

## 📜 Project Structure

- `src/components`: Reusable UI components (Navbar, Footer, etc.)
- `src/pages`: Application views (Landing, Dashboard, History, etc.)
- `src/context`: Authentication and global state management
- `src/services`: Mock API layers for trip generation
- `src/types.ts`: TypeScript interfaces and shared types
